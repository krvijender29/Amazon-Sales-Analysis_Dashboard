# Amazon Sales Analysis Dashboard

## Project Overview

This project analyzes Amazon product sales data and provides useful insights through an interactive Streamlit dashboard.

The dashboard helps users explore:

* Product pricing trends
* Discount analysis
* Product ratings
* Customer reviews
* Category-wise performance
* Top-performing products

## Dataset

The dataset contains information about:

* Product Name
* Category
* Actual Price
* Discounted Price
* Discount Percentage
* Product Rating
* Rating Count
* Customer Reviews

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Streamlit

## Features

* KPI Metrics
* Top 10 Highest Rated Products
* Top 10 Most Reviewed Products
* Category-wise Analysis
* Discount Analysis
* Price Distribution
* Rating vs Discount Analysis

## Project Goal

The objective of this project is to perform exploratory data analysis (EDA) on Amazon product data and visualize key business insights using an interactive dashboard.
